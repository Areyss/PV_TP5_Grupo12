## 🧠 Descripción del proyecto

Este proyecto es una aplicación de gestión de alumnos desarrollada en React con Vite como parte del Trabajo Práctico N°5 de la materia **Programación Visual**.  
El objetivo principal es aplicar el uso de **eventos en React**, junto con la implementación de **React Router** para la navegación entre vistas.

La aplicación permite agregar, editar, eliminar y visualizar datos de alumnos, utilizando componentes funcionales y una estructura organizada que facilita la escalabilidad del proyecto.

### 🛠 Funcionalidades implementadas

- 🏠 **Home**: Vista de bienvenida o presentación del sistema.
- 📋 **Lista de alumnos**: Muestra todos los alumnos en forma de tabla o tarjetas.
- ➕ **Agregar alumno**: Formulario para crear un nuevo alumno.
- 🖊 **Editar alumno**: Formulario precargado para modificar un alumno existente.
- 🔍 **Detalle del alumno**: Vista con toda la información detallada del alumno.
- 🗺 **Navegación general**: Incluye enlaces hacia todas las secciones de la aplicación.

La aplicación fue creada utilizando **Vite** con la plantilla de React, y organizada en componentes reutilizables para una mejor estructura.
